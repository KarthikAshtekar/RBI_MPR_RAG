# Retrieval Configuration Selection Policy

Written before combined-candidate or held-out evaluation.

1. Single-report contamination must be zero.
2. Valid comparative report coverage must be 100%.
3. Maximise complete evidence recall.
4. Break ties using all-reports hit, evidence recall, macro report MRR, median latency, then simplicity.
5. Median latency should not exceed twice the frozen development baseline unless evidence gains are substantial.
6. Held-out results cannot alter selection.
